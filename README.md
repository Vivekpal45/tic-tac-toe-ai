# Tic Tac Toe AI

An interactive Tic Tac Toe game built with Python and Tkinter featuring an unbeatable AI powered by the Minimax Algorithm.

## Features

* Interactive GUI using Tkinter
* Choose your symbol (X or O)
* Unbeatable AI using Minimax Algorithm
* Real-time score tracking
* Games played counter
* Restart current game
* Start a new match
* Quit option with score reset
* User-friendly interface

## Technologies Used

* Python
* Tkinter
* Minimax Algorithm
* Game Theory


## How It Works

The AI uses the Minimax Algorithm to evaluate all possible game states and always selects the optimal move. Because Tic Tac Toe has a limited search space, the AI can calculate the best move instantly, making it impossible to defeat.
